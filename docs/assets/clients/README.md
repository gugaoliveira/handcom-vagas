# Logos de Clientes

## Como adicionar logos reais

1. Adicione arquivos de imagem nesta pasta com os nomes:
   - `gerdau.png` ou `gerdau.jpg`
   - `andrade-gutierrez.png` ou `andrade-gutierrez.jpg`
   - `guanabara.png` ou `guanabara.jpg`
   - `bahamas.png` ou `bahamas.jpg`
   - `abc-construcao.png` ou `abc-construcao.jpg`
   - `fgv.png` ou `fgv.jpg`

2. Edite `index.html` e substitua as divs `.client-logo-text` por:
   ```html
   <img src="assets/clients/gerdau.png" alt="Gerdau">
   ```

## Formato recomendado
- **Formato:** PNG com fundo transparente (preferencial) ou JPG
- **Tamanho:** Máximo 400x200px
- **Peso:** Menos de 100KB por logo
- **Estilo:** Logo oficial da empresa, preferencialmente horizontal

## Atualmente
O portal usa placeholders de texto até que as logos sejam adicionadas.
